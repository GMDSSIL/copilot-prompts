# copilot-prompts - Atividade (IA Mentor de Carreira: Descubra Seu Futuro em Tech- DIO)

💻 Sobre o Projeto  
Vamos construir uma Single Page Application (SPA) responsiva, voltada para atender diferentes tipos de micro, pequenas e médias empresas.
A página conterá as seguintes seções: Header, Navigation, Home, Sobre, Serviços, Depoimentos, Contato e Footer, que será responsável por centralizar informações de copyright e links úteis.

📚 Pré-requisitos de Habilidades e Níveis de Conhecimento  
Antes de ingressar neste conteúdo, é necessário possuir conhecimento prévio nas seguintes áreas:

Lógica de programação | Básico

HTML | Básico

CSS | Básico

JavaScript | Básico

Gerenciamento de pacotes (npm/yarn) | Básico

🛠️ Habilidades e Sub-habilidades que vamos aprender neste conteúdo

Principais habilidades:

Desenvolvimento Web Frontend
